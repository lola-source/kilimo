# Kilimo

Kilimo is a Django-based web application, the project is containerized using Docker and includes CI/CD integration with GitHub Actions.


## Features

- User authentication


## Technologies Used

- Django
- Docker
- PostgreSQL
- GitHub Actions
- Flake8 (for linting)
